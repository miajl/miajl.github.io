My Bachelorâ€™s capstone project was implementing several autonomy functionalities on a robot car. The project was done in teams and includes localization with a particle filter, path planning with RRT*, control with a pure pursuit controller, and obstacle avoidance. The project was implemented in Python using ROS and involved camera and LiDAR sensors.
